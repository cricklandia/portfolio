Case Note System

Vue, JavaScript, Django, Docker
Spring 2022

This was the final project for my software engineering course. My team was responsible for designing, implementing, and testing the front end requirements given to us by the customer. With diligent communication with the client and my team, we were able to develop a reliable product which pushed, posted, rendered, and updated data between various user views and established connections between the front and back end. The front end aspects that I worked on were written in Vue.js and plain JavaScript. Other aspects of the project included a Docker container environment consisting of NGINX, Django, and Vue servers. Python was used for terminal commands and scripts to manage the back end as well as to define data tables within the Django REST API. Version control was all done through GitHub using terminal commands and the Visual Studio Code integrated source control.

Learn-to-Read

Wordpress Website
2020

As part of an Athens State course called Programming for the Web, my assigned client was a a nonprofit literacy organization whose existing website was outdated and needed to be redesigned and updated. My partner and I created an option for the client using Wordpress. It featured responsive resizing (not shown in video) as well as a widget showing the organization's Facebook activity, a Google map widget, and a migration of all content from their previous website.
